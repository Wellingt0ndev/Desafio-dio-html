As tags h1 até a h6

As tags de cabeçalho em HTML, representadas pelos Tags h1 até h6, servem para estruturar o conteúdo de uma página em diferentes níveis de hierarquia. Aqui estão algumas informações importantes sobre essas tags:

  1. Hierarquia de Títulos:
    • h1 é o nível de seção mais alto, usado para o título principal da página.
    • h2 é um nível abaixo do h1, adequado para subtítulos importantes.
    • h3 é um nível abaixo do h2, e assim por diante até o h6.
    • h6 é o nível de seção mais baixo.
  2. Utilização:
    • Essas tags ajudam a organizar o conteúdo e indicam a importância relativa dos títulos e subtítulos.
    • Agentes de usuário (como navegadores) podem usar essas informações para construir tabelas de conteúdo automaticamente.

Tag p

A tag p é utilizada para demarcar um parágrafo. A tag HTML p representa um parágrafo do texto. Parágrafos são, geralmente, representados em mídia visual como blocos de texto separados dos blocos adjacentes por espaços brancos verticais e/ou recuo de primeira linha

Tag mark

A tag mark é usado para destacar partes do texto. Ele representa um trecho de destaque em um contexto específico, como uma sequência de texto marcada como referência ou relevante. Por exemplo, em uma página de resultados de busca, todas as instâncias da palavra pesquisada podem ser destacadas com A tag mark.

Tag small

A tag small é usado para destacar partes do texto. Ele representa um trecho de destaque em um contexto específico, como letras pequenas de exoneramentos, salvedades, avisos legais, declarações de direitos autorais ou atribuições. Por padrão, o texto dentro desse Tag é renderizado com um tamanho de fonte menor, como de pequeno para x-pequeno

Tags i, u e strong

  1. Tag i
    • A tag i é usado para aplicar estilo itálico ao texto.
    • Ele não implica significado semântico, mas é usado para ênfase visual.
  2. Tag u
    • A tag u é usado para aplicar estilo de sublinhado ao texto.
    • No entanto, seu uso é desencorajado, pois o sublinhado geralmente indica um link.
  3. Tag strong
    • A tag strong dá ao texto uma forte importância e é tipicamente mostrado em negrito.
    • É usado para destacar partes do texto que são de grande importância.
    • Não deve ser usado apenas para aplicar estilo negrito; use CSS (font-weight) para isso.

Tag a

A tag a em HTML (ou elemento âncora) cria um hiperlink para outras páginas da web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outro destino que uma URL possa indicar. O atributo mais importante do elemento a é o href, que especifica o destino do link.

Tag hr

A tag hr em HTML representa uma quebra temática entre elementos de nível de parágrafo. Ela é frequentemente exibida como uma linha horizontal que separa ou define uma mudança no conteúdo de uma página HTML

Tag sub

A tag sub é usada para definir texto subscrito em um documento HTML. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. Essa tag é frequentemente usada para representar fórmulas químicas, como H2O, onde o “2” está subscrito.

Tag sup

A tag sup é usada para definir texto sobrescrito em um documento HTML. O texto sobrescrito aparece meio caractere acima da linha normal e é frequentemente usado para representar notações matemáticas, como expoentes ou índices.

a2 + b2 = c2

Tag blockquote

A tag blockquote em HTML (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, esse texto é processado visualmente com recuo . Você pode fornecer a URL da fonte da citação usando o atributo cite, ou uma representação de texto da fonte usando o elemento cite.

Tag font

A tag font era usada no HTML 4 para especificar a fonte, o tamanho da fonte e a cor do texto. No entanto, o uso dessa tag está desencorajado e não é mais suportado no HTML5. Em vez disso, a formatação de fonte, cor e tamanho é feita através do CSS.

Tag del

A tag del é usada para identificar o texto que foi excluído de um documento, mas retido para mostrar o histórico de modificações feitas no documento. Ela é frequentemente usada em revisões de texto ou comparações de versões. Quando usada, o texto dentro da tag del é geralmente exibido com uma linha horizontal através dele para indicar que foi removido.

Tag abbr

A tag abbr (ou Elemento de Abreviação HTML) é usada para representar uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela. Essa tag é útil para fornecer informações úteis para navegadores, corretores ortográficos, sistemas de tradução e indexadores.

Obama é presidente dos EUA.
